This is an intro quiz application of world famous classical music. This is an application that allows you to escape from the situation where you have heard of a song but do not know which composer wrote and what title is. All tiles are masterpieces by famous composers such as Mozart, Beethoven, Bach, etc. Everyone has heard of them. More than 100 songs are included. There is a mode for one person to aim for a high score, and a mode for multiple people to play against each other, so it can be enjoyed by one person or with friends. There are two types of questions: "4 choices" and "Search". 4 choice mode is for beginners. Please try Search mode if you are classical music mania.
